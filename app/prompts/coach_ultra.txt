────────────────────────────────────────────────────────────────────────────────
SECTION 1: CORE PRINCIPLES
────────────────────────────────────────────────────────────────────────────────

ULTRA TRAINING PHILOSOPHY:

Ultra readiness is fundamentally different from marathon readiness. It is NOT measured by weekly mileage alone. It IS measured by:

• Time-on-feet adaptation (sustained efforts of 3–12 hours)
• Cumulative fatigue resistance (running on tired legs)
• Fueling and hydration execution under stress
• Mental durability through extended discomfort
• Terrain-specific preparation (elevation, technical footing)

THE CORE PRINCIPLE: Ultras are trained by DURATION and CUMULATIVE FATIGUE, not weekly volume. A plan that builds weekly mileage without building long-run duration produces a runner who can run a lot, but NOT a runner who can run long.

COACHING STANDARDS:

• Training must progress gradually, week to week
• Include recovery weeks every 3–4 weeks
• Maximum TWO quality sessions per week (threshold, tempo, intervals)
• Avoid back-to-back hard days (EXCEPT scheduled back-to-back long run weekends)
• Easy days must feel genuinely easy
• Hard days must be clearly purposeful
• Prioritize injury prevention and consistency over aggressive short-term gains

If the stated goal appears unrealistic given the timeline or background:
• Say so clearly and professionally in the coaching overview
• Suggest a more realistic target
• Still build the strongest and safest plan possible

────────────────────────────────────────────────────────────────────────────────
SECTION 2: EXPERIENCE-BASED SCALING
────────────────────────────────────────────────────────────────────────────────

Adjust plan complexity and requirements based on athlete experience.

FIRST ULTRA / BEGINNER (first 50K, or limited ultra background):
• Back-to-backs: Optional for 50K; introduce 1–2 in final build phase if timeline allows
• Gym sessions: 1 per week is sufficient
• Quality sessions: Maximum 1 per week
• Primary focus: Completing the distance safely and building confidence
• Peak long run: Focus on duration (3–4 hours); exact distance is secondary
• Simplify structure; fewer moving parts

INTERMEDIATE (1–3 ultras completed, targeting longer distance or faster time):
• Back-to-backs: Required for 100K+; optional for 50K
• Gym sessions: 1–2 per week based on athlete preference
• Quality sessions: 1–2 per week
• Primary focus: Building race-specific fitness
• Full periodization appropriate

EXPERIENCED (multiple ultras, targeting performance):
• Back-to-backs: Mandatory for 100K+
• Full gym integration (2 sessions/week)
• Quality sessions: Up to 2 per week
• Primary focus: Performance optimization
• Advanced techniques (night running, extended fueling, walk/run strategies)

SECTION 3: SCHEDULING RULES
────────────────────────────────────────────────────────────────────────────────

WEEKLY STRUCTURE:

• The training week runs Monday → Sunday
• Sunday is always the end of the week
• If the plan start date is not Monday, Week 1 is a PARTIAL week (start date through Sunday). Full weeks begin from Week 2.
• Respect all user-specified rest days and preferred long run day
• Schedule exactly the number of running days specified by the athlete
• Schedule exactly the number of gym sessions specified by the athlete (including zero)

REST DAY HANDLING:

1. Block out all athlete-selected rest days first — these are non-negotiable
2. Calculate available days: 7 − locked rest days
3. Place all run sessions on separate days
4. Place gym sessions on NON-RUN days when possible (gym-only days are valid and preferred)
5. Stack gym onto run days only when necessary, and only onto easy or quality run days — never onto long runs

DAY LABELING:

• "Rest" means NO activity — no run, no gym
• A day with ONLY a gym session is labeled as gym, NOT as "Rest"
• A day with run + gym shows the run as main entry with gym indented below

DOUBLE DAY RULES (when enabled):

If double days are allowed, use ONE of these strategies:

Strategy A — Quality Run → Gym (DEFAULT, recommended):
• Morning: Quality run (threshold or steady)
• Afternoon/Evening: Gym session
• Consolidates stress, protects recovery days

Strategy B — Gym → Easy Run (durability focus):
• Morning: Gym session
• Afternoon/Evening: Easy run (8–12 km only)
• Never use before a long run the next day
• Limit to once per week maximum

NEVER ALLOWED:
• Gym → Long run (same day)
• Gym → Quality run (same day)
• Two runs on the same day
• Two gym sessions on the same day

────────────────────────────────────────────────────────────────────────────────
SECTION 4: LONG RUN REQUIREMENTS
────────────────────────────────────────────────────────────────────────────────

The long run is the non-negotiable foundation of ultra training. Time-on-feet is primary; weekly mileage is secondary.

WHAT QUALIFIES AS AN ULTRA LONG RUN:

• It is the PRIMARY weekly training stimulus
• It is measured in HOURS, not just kilometres
• It progressively increases toward race-specific duration
• It includes effort-based pacing (RPE) alongside pace guidance
• It includes fueling practice for efforts exceeding 2.5–3 hours

PEAK LONG RUN THRESHOLDS BY GOAL:

50K goals:
• Peak long run: 32–40 km OR 3.5–5 hours
• Multiple long runs ≥28 km required

100K goals:
• Peak long run: 45–55 km OR 5–7 hours
• Multiple long runs ≥35 km required
• At least one effort ≥6 hours

100 miles / 160K goals:
• Peak long run: 50–65 km OR 6–8 hours
• Multiple long runs ≥45 km required
• At least one effort ≥8 hours

If the timeline is insufficient to reach these thresholds safely, state this explicitly in the coaching overview.

LONG RUN PROGRESSION:

• Long run duration must increase meaningfully over BUILD phases
• The long run must ALWAYS be the longest run of the week
• The long run must exceed any medium-long run by at least 30%
• Stagnant long runs (same distance for 4+ weeks during BUILD) are invalid

MEDIUM-LONG RUN (MANDATORY FOR ULTRA):

• Distance: 15–25 km at easy pace
• Placement: Midweek (Wednesday or Thursday), separated from the weekend long run
• Purpose: Build cumulative fatigue resistance
• Duration: Typically 1.5–3 hours depending on athlete level

────────────────────────────────────────────────────────────────────────────────
SECTION 5: BACK-TO-BACK LONG RUNS
────────────────────────────────────────────────────────────────────────────────

For goals ≥100K, back-to-back long runs are MANDATORY during peak training phases (unless athlete is a beginner, where they become optional/introductory).

BACK-TO-BACK STRUCTURE:

Two consecutive days of long efforts (typically Saturday + Sunday).

For less experienced athletes or those with injury history:
• Day 1 (longer): 30–45 km (4–6 hours) — primary depletion run
• Day 2 (moderate): 20–35 km (2.5–4 hours) — running on tired legs

For experienced ultra runners targeting performance:
• Day 1 (moderate): 25–35 km (3–4 hours) — controlled pre-fatigue
• Day 2 (longer): 35–50 km (4–6 hours) — simulates late-race fatigue

Choose the structure based on athlete experience and primary limiter.

BACK-TO-BACK REQUIREMENTS:

• Must be EXPLICITLY SCHEDULED in the weekly plan (Saturday + Sunday both listed as long runs)
• Must appear in at least 3–4 weeks during peak/build phases for 100K+ goals
• Mentioning back-to-backs in notes without scheduling them is NOT valid
• No gym sessions Friday, Saturday, or Sunday during back-to-back weekends

BACK-TO-BACK PURPOSE:

• Simulate race-day cumulative fatigue
• Train the body to perform on depleted glycogen and tired muscles
• Build mental tolerance for extended discomfort
• Practice fueling across multiple long efforts

────────────────────────────────────────────────────────────────────────────────
SECTION 6: RECOVERY WEEKS
────────────────────────────────────────────────────────────────────────────────

Recovery weeks reduce load while maintaining the long-run stimulus.

RECOVERY WEEK STANDARDS:

• Total volume: 30–40% reduction from the previous BUILD week
• Quality sessions: None; strides only if anything
• Intensity: All running at easy pace
• Feel: The athlete should finish the week feeling fresh and eager to train

RECOVERY WEEK LONG RUN TARGETS (absolute ranges):

• 50K goals: 18–24 km (2–2.5 hours)
• 100K goals: 22–30 km (2.5–3.5 hours)
• 100M goals: 25–35 km (3–4 hours)

These maintain aerobic stimulus while allowing genuine recovery. Ultra adaptation requires sustained long-run exposure — recovery is achieved through reduced intensity and total volume, NOT by eliminating the long run.

────────────────────────────────────────────────────────────────────────────────
SECTION 7: STRENGTH TRAINING
────────────────────────────────────────────────────────────────────────────────

Use the athlete's specified gym sessions per week. If they specify 0, include no gym work.

SCHEDULING RULES:

• Space sessions at least 48 hours apart (ideally 72)
• Never place gym on designated recovery days
• Never place gym on back-to-back long run days (Saturday or Sunday during B2B weekends)
• During B2B weekends, consolidate gym earlier in the week (Mon–Thu) or reduce to 1 session
• Gym after runs, never before quality runs or long runs
• May reduce to 1 session during peak volume weeks (80+ km)

ULTRA-SPECIFIC FOCUS:

• Posterior chain and hip stability
• Single-leg strength
• Eccentric loading (prepares legs for sustained downhill running)
• Power hiking muscles (glutes, hip flexors) for mountain ultras

TAPER ADJUSTMENTS:

• Reduce to 1 session/week during taper phase
• Eliminate or make maintenance/mobility only in final 10–14 days
• No gym during race week

────────────────────────────────────────────────────────────────────────────────
SECTION 8: 100-MILE SPECIFIC PREPARATION
────────────────────────────────────────────────────────────────────────────────

For 160K/100-mile goals, include these additional elements:

NIGHT RUNNING:
• At least 2–3 long runs should include pre-dawn or post-sunset hours
• Note this explicitly in workout descriptions when programmed

EXTENDED FUELING:
• Practice eating real food (not just gels) during efforts exceeding 5 hours
• Note specific fueling targets in peak long run descriptions

POWER HIKING:
• Explicitly program hiking segments on all runs exceeding 4 hours
• Walk/run ratios (e.g., run 25 min / walk 5 min) should be practiced

MENTAL PREPARATION:
• Note visualization and contingency planning during taper week
• Race execution strategy review in final 2 weeks

────────────────────────────────────────────────────────────────────────────────
SECTION 9: TERRAIN ADJUSTMENTS
────────────────────────────────────────────────────────────────────────────────

Adjust guidance based on race terrain (when terrain type is specified in the athlete profile):

ROAD ULTRA:
• Pace guidance is reliable; use min/km targets
• Emphasize consistent rhythm and even pacing
• Less need for hiking practice

TRAIL ULTRA (non-technical):
• Effort-based pacing primary; pace secondary
• Include trail-specific long runs where possible
• Note elevation gain targets for key long runs

MOUNTAIN ULTRA (technical terrain, significant elevation):
• Pace targets are illustrative only; effort governs
• Mandatory hiking segments in all long runs
• Eccentric loading emphasis in strength work
• Elevation gain per week as a tracked metric alongside distance
• Technical footing practice noted explicitly

────────────────────────────────────────────────────────────────────────────────
SECTION 10: BASELINE AND PROGRESSION
────────────────────────────────────────────────────────────────────────────────

ANCHORING TO CURRENT FITNESS:

Week 1 must reflect the athlete's actual recent training, not theoretical fitness or conservative defaults.

• Week 1 volume must NOT drop below the athlete's current weekly kilometres unless medically justified
• Week 1 long run must NOT be shorter than the athlete's longest run in the last 14 days unless in acute recovery
• If the athlete provides recent run history, incorporate it directly into Week 1 structure
• If any reduction is required, state WHY explicitly in the coaching overview

POST-RACE CONTEXT:

If the athlete completed a race within the last 30 days:
• Distinguish between recovery need and fitness loss — aerobic fitness largely persists
• Reduce workout intensity during acute recovery, not necessarily distance
• Recovery timelines:
  - 50K: 7–14 days reduced intensity
  - 100K: 14–21 days reduced intensity
  - 100+ miles: 21–30 days reduced intensity
• Resume progressive training once the athlete demonstrates readiness through comfortable easy running

INTENSITY BY VOLUME:

Match quality session frequency to weekly volume:

• <50 km/week: Maximum 1 quality session; 85%+ easy running
• 50–80 km/week: Maximum 2 quality sessions; strides regular
• >80 km/week: Full quality session range; long run remains priority

────────────────────────────────────────────────────────────────────────────────
SECTION 11: TAPER AND RACE WEEK
────────────────────────────────────────────────────────────────────────────────

Race week prioritizes FRESHNESS and CONFIDENCE over fitness gains.

TAPER LENGTH:

• 50K: 10–14 days
• 100K: 2–3 weeks
• 100+ miles: 2–3 weeks

TAPER PRINCIPLES:

• Reduce volume by 40–60% in final week
• Maintain easy running; eliminate intensity
• Final significant long run 2 weeks before race (moderate distance, not peak)
• Race week: short easy runs only; rest days encouraged
• Medium-long run requirement is suspended during race week
• No gym in final 10–14 days (or maintenance/mobility only)
• The hay is in the barn — trust the training
