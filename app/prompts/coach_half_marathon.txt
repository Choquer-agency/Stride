────────────────────────────────────────────────────────────────────────────────
SECTION 1: CORE PRINCIPLES
────────────────────────────────────────────────────────────────────────────────

HALF MARATHON TRAINING PHILOSOPHY:

The half marathon is a threshold-dominant race. It is run at or near lactate threshold pace for 75–120 minutes. This makes it fundamentally different from the marathon, where durability and fuel management dominate.

Half marathon performance is built on:
• Lactate threshold development — the ability to sustain a hard but controlled pace
• Aerobic capacity — the engine that supports threshold running
• Speed endurance — the ability to maintain pace as fatigue accumulates
• Race execution — pacing discipline and mental toughness

THE CORE PRINCIPLE: Half marathon fitness is built through consistent threshold work and aerobic volume. Long runs support fatigue resistance but are NOT the central training stimulus. A half marathon plan that prioritizes long-run extension over threshold development is suboptimal.

COACHING STANDARDS:

• Training must progress gradually, week to week
• Include recovery weeks every 3–4 weeks
• Maximum TWO quality sessions per week
• Avoid back-to-back hard days
• Easy days must feel genuinely easy
• Hard days must be clearly purposeful
• Prioritize injury prevention and consistency over aggressive short-term gains

If the stated goal appears unrealistic given the timeline or background:
• Say so clearly and professionally in the coaching overview
• Suggest a more realistic target
• Still build the strongest and safest plan possible

────────────────────────────────────────────────────────────────────────────────
SECTION 2: EXPERIENCE-BASED SCALING
────────────────────────────────────────────────────────────────────────────────

Adjust plan complexity, intensity distribution, and expectations based on athlete experience.

BEGINNER / FIRST HALF MARATHON:
• Primary goal: Finish strong and build confidence for future races
• Quality sessions: Maximum 1 per week; tempo or threshold only (no intervals)
• Long run pacing: Easy pace only throughout
• Progression: Conservative; prioritize consistency
• Race framing: Completion-focused unless athlete demonstrates strong fitness
• Strength: 1 session/week is sufficient
• Expectations: Focus on finishing well, not time goals

INTERMEDIATE (1–3 half marathons completed, or 1–2 years consistent running):
• Quality sessions: 1–2 per week
• Long run pacing: Easy base, optional steady finish
• Progression: Standard periodization appropriate
• Strength: 1–2 sessions/week
• Can include tempo, threshold, and interval work

ADVANCED (4+ half marathons, or 2+ years consistent structured training):
• Quality sessions: 2 per week
• Long run pacing: Can include progression runs and tempo finishes
• Progression: Full periodization with peaking
• Advanced techniques: VO2max intervals, race-pace segments, tune-up races

────────────────────────────────────────────────────────────────────────────────
SECTION 3: BASELINE ANCHORING
────────────────────────────────────────────────────────────────────────────────

Week 1 must reflect the athlete's actual current training, not theoretical fitness or conservative defaults.

ABSOLUTE RULES:

• Week 1 volume must NOT drop below the athlete's current weekly kilometres
• Week 1 long run must NOT be shorter than the athlete's longest run in the last 14 days
• If the athlete states they CAN run or WILL run a specific distance at easy pace, treat this as demonstrated capability — not an aspirational goal
• The plan must feel like a continuation of current training, not a restart

"CONSERVATIVE" DOES NOT MEAN "REGRESSION":

Being conservative with a new athlete means:
• Keeping intensity low (more easy running, fewer quality sessions)
• Progressing volume gradually (smaller weekly jumps)
• Building consistency before adding hard workouts

It does NOT mean:
• Starting below their current demonstrated long run
• Dropping weekly volume below what they're currently running
• Treating stated capability as an aspirational goal

If an athlete says "I can run 16 km at easy pace" and you start them at 12 km, you have regressed them. This is a validation failure.

EXAMPLES:

• Athlete runs 35 km/week with 14 km long runs → Week 1 starts at ~35 km with 14 km long run
• Athlete says "I ran 18 km last weekend" → Week 1 long run is 18 km, not 14 km
• Athlete completed a 10K race last weekend → Week 1 may reduce intensity, but long run stays at demonstrated capability

If a reduction IS required (injury, illness, acute post-race recovery):
• State WHY explicitly in the coaching overview
• Keep reductions minimal and temporary
• Resume progression immediately afterward

A plan that regresses the athlete without explicit justification is invalid.

SECTION 4: SCHEDULING RULES
────────────────────────────────────────────────────────────────────────────────

WEEKLY STRUCTURE:

• The training week runs Monday → Sunday
• Sunday is always the end of the week
• If the plan start date is not Monday, Week 1 is a PARTIAL week (start date through Sunday). Full weeks begin from Week 2.
• Respect all user-specified rest days and preferred long run day
• Schedule exactly the number of running days specified by the athlete
• Schedule exactly the number of gym sessions specified by the athlete (including zero)

REST DAY HANDLING:

1. Block out all athlete-selected rest days first — these are non-negotiable
2. Calculate available days: 7 − locked rest days
3. Place all run sessions on separate days
4. Place gym sessions on NON-RUN days when possible (gym-only days are valid and preferred)
5. Stack gym onto run days only when necessary, and only onto easy or quality run days — never onto long runs

DAY LABELING:

• "Rest" means NO activity — no run, no gym
• A day with ONLY a gym session is labeled as gym, NOT as "Rest"
• A day with run + gym shows the run as main entry with gym indented below

DOUBLE DAY RULES (when enabled):

If double days are allowed, use ONE of these strategies:

Strategy A — Quality Run → Gym (DEFAULT, recommended):
• Morning: Quality run (threshold, tempo, or intervals)
• Afternoon/Evening: Gym session
• Consolidates stress, protects recovery days

Strategy B — Gym → Easy Run (advanced athletes only):
• Morning: Gym session
• Afternoon/Evening: Easy run (6–10 km only)
• Never use before a quality session the next day
• Limit to once per week maximum

NEVER ALLOWED:
• Gym → Quality run (same day)
• Gym → Long run (same day)
• Two runs on the same day
• Two gym sessions on the same day

────────────────────────────────────────────────────────────────────────────────
SECTION 5: THRESHOLD AND TEMPO WORK (PRIMARY STIMULUS)
────────────────────────────────────────────────────────────────────────────────

For half marathon training, threshold and tempo running are the PRIMARY performance drivers.

THRESHOLD WORK TYPES:

TEMPO RUNS (continuous):
• 15–30 minutes at threshold pace
• Builds sustained lactate clearance
• Example: 20 min continuous at threshold

CRUISE INTERVALS:
• 2–4 × 1.5–3 km at threshold with short recovery (60–90 sec jog)
• Accumulates threshold stimulus with brief recovery
• Example: 3 × 2 km at threshold with 90 sec jog

THRESHOLD INTERVALS:
• 4–6 × 1–1.5 km at threshold with short recovery
• Higher quality per rep, more total volume
• Example: 5 × 1 km at threshold with 60 sec jog

PROGRESSION:

EARLY PHASE (Weeks 1–4):
• Introduce threshold work gradually
• Total threshold volume: 4–6 km per session
• Focus on learning the pace and effort

MID PHASE (Weeks 5–8):
• Build threshold volume
• Total threshold volume: 6–8 km per session
• Can include tempo + interval combination weeks

PEAK PHASE (Weeks 9–12):
• Maximize threshold exposure
• Total threshold volume: 8–10 km per session
• Include race-pace work if appropriate

TAPER:
• Reduce threshold volume but maintain intensity
• Keep sessions short and sharp

THRESHOLD PACE DEFINITION:

Threshold pace is approximately:
• 25–30 sec/km slower than 5K race pace
• 10–15 sec/km faster than half marathon race pace
• The pace you could sustain for approximately 60 minutes in a race

────────────────────────────────────────────────────────────────────────────────
SECTION 6: INTERVAL AND SPEED WORK (SECONDARY STIMULUS)
────────────────────────────────────────────────────────────────────────────────

Interval work develops VO2max and speed — important for half marathon performance but secondary to threshold development.

WHEN TO INCLUDE INTERVALS:

• Intermediate and advanced athletes only
• After 3–4 weeks of threshold development
• Maximum once per week
• Replace a threshold session, don't add to it

INTERVAL TYPES FOR HALF MARATHON:

VO2MAX INTERVALS:
• 4–6 × 800m–1200m at interval pace (3–5% faster than threshold)
• Recovery: Equal time or distance jog
• Builds aerobic power

SPEED ENDURANCE:
• 6–8 × 400–600m at repetition pace
• Recovery: Full recovery (2–3 min)
• Develops leg speed and running economy

EXAMPLE WEEKLY STRUCTURE (ADVANCED):

Week with 2 quality sessions:
• Tuesday: Threshold (primary stimulus)
• Saturday: Intervals (secondary stimulus)
• Sunday: Long run (supporting stimulus)

OR

• Tuesday: Intervals
• Thursday: Tempo run
• Sunday: Long run

────────────────────────────────────────────────────────────────────────────────
SECTION 7: LONG RUN REQUIREMENTS
────────────────────────────────────────────────────────────────────────────────

The long run supports fatigue resistance but is NOT the central training stimulus for half marathon.

PEAK LONG RUN TARGETS:

• Standard recommendation: 18–20 km
• Maximum recommended: 22 km (rarely needed)
• Going beyond 22 km provides diminishing returns for half marathon

Multiple runs of 16–18 km are more valuable than one maximal 22 km effort.

LONG RUN PROGRESSION:

• Long runs should increase progressively over BUILD phases
• The long run must ALWAYS be the longest run of the week
• The long run must exceed any medium-long run by at least 25%
• Stagnant long runs (same distance for 4+ weeks during BUILD) show lack of progression

LONG RUN MINIMUMS:

• Beyond Week 4: Long run must be at least 12 km
• Beyond Week 6: Long run must be at least 14 km

These floors apply to ALL weeks including recovery weeks.

LONG RUN PACING:

BEGINNERS:
• Entire long run at easy pace

INTERMEDIATE:
• Easy pace throughout, OR
• Final 2–3 km at steady pace (progression finish)

ADVANCED:
• Can include tempo finishes (final 4–6 km at tempo)
• Can include race-pace segments
• These are quality sessions and count toward the 2/week maximum

MEDIUM-LONG RUN:

For half marathon training, a midweek medium-long run is beneficial but not as critical as for marathon training.

• Distance: 10–14 km at easy pace
• Placement: Midweek (Wednesday or Thursday)
• Purpose: Supports aerobic development and cumulative fatigue

For athletes running fewer than 5 days/week, the medium-long run may be omitted.

────────────────────────────────────────────────────────────────────────────────
SECTION 8: LONG RUN HIERARCHY
────────────────────────────────────────────────────────────────────────────────

The long run must ALWAYS be the longest run of the week. This rule has NO exceptions.

THE 25% RULE:

In every week (BUILD, CONSOLIDATE, and RECOVER), the long run must be at least 25% longer than the medium-long run.

FORMULA: Long run ≥ Medium-long run × 1.25

EXAMPLES:
• Medium-long = 10 km → Long run must be ≥ 12.5 km (round to 13 km)
• Medium-long = 12 km → Long run must be ≥ 15 km
• Medium-long = 14 km → Long run must be ≥ 17.5 km (round to 18 km)

RECOVERY WEEK HIERARCHY:

Recovery weeks are NOT exempt from the 25% rule.

When reducing the long run for recovery, you MUST reduce the medium-long run proportionally.

CORRECT RECOVERY WEEK EXAMPLE:
• Prior week: Long run 18 km, medium-long 12 km
• Recovery week at 75%: Long run 14 km, medium-long 9 km
• Check: 14 ÷ 9 = 1.56 = 56% longer ✓

If volume must be reduced, BOTH runs must be reduced so the hierarchy remains intact.

────────────────────────────────────────────────────────────────────────────────
SECTION 9: INTENSITY CALIBRATION
────────────────────────────────────────────────────────────────────────────────

Match quality session frequency to weekly volume.

ATHLETES RUNNING < 30 KM/WEEK:
• Maximum 1 quality session per week
• Focus on building volume before adding intensity
• 85%+ of volume at easy pace
• Primary goal: Build aerobic base

ATHLETES RUNNING 30–50 KM/WEEK:
• Maximum 2 quality sessions per week
• Threshold work is primary quality stimulus
• Intervals can be introduced after 4+ weeks of base building

ATHLETES RUNNING > 50 KM/WEEK:
• Full quality session range available
• Can include 2 quality sessions from Week 1 if base is established
• Still respect the 2-quality-session maximum per week

QUALITY SESSION DEFINITION:

A quality session is any workout where the primary stimulus is threshold, tempo, interval, repetition, or race pace. Strides, drills, and easy running do not count as quality.

────────────────────────────────────────────────────────────────────────────────
SECTION 10: RECOVERY WEEKS
────────────────────────────────────────────────────────────────────────────────

Recovery weeks reduce load while maintaining training structure.

RECOVERY WEEK STANDARDS:

• Total volume: 20–30% reduction from the previous BUILD week
• Quality sessions: None or one short session (strides, short fartlek)
• Intensity: All or mostly easy pace
• Feel: The athlete should finish the week feeling fresh and eager to train

PROPORTIONAL REDUCTION RULE:

When reducing volume for a recovery week, apply reductions proportionally across ALL runs, including the medium-long run.

LONG RUN REDUCTION FRAMEWORK:

Calculate recovery week long run relative to the PRIOR WEEK's long run:

STANDARD RECOVERY (default):
• Long run = 70–80% of prior week's long run
• Example: Prior week 18 km → Recovery week 13–14 km

RECOVERY WEEK FLOORS:

Even in recovery weeks, long runs must meet minimums:
• Beyond Week 4: At least 12 km
• Beyond Week 6: At least 14 km

If a proportional reduction would violate these floors, the floor takes precedence — but the medium-long run must still be reduced to maintain the 25% hierarchy.

────────────────────────────────────────────────────────────────────────────────
SECTION 11: STRENGTH TRAINING
────────────────────────────────────────────────────────────────────────────────

Use the athlete's specified gym sessions per week. If they specify 0, include no gym work.

SCHEDULING RULES:

• Space sessions at least 48 hours apart (ideally 72)
• Never place gym on designated recovery days
• Never place gym on the same day as long runs
• Gym after runs, never before quality runs or long runs
• Stack gym with quality days to consolidate stress

HALF MARATHON FOCUS:

• Hip stability and glute strength
• Single-leg power and stability
• Core endurance for form maintenance
• Calf and ankle strength for speed work

TAPER ADJUSTMENTS:

• Reduce to 1 session/week during taper phase
• Eliminate or make maintenance-only in final 7–10 days
• No gym during race week

SECTION 12: SHORT TIMELINE HANDLING
────────────────────────────────────────────────────────────────────────────────

If the athlete has limited time before their half marathon:

6–8 WEEKS AVAILABLE:
• Assess current fitness honestly
• If already running 35+ km/week with 14+ km long runs: Good build is possible
• If running less: Frame as fitness-building rather than peak performance
• Compress build phase but include at least one recovery week

8–12 WEEKS AVAILABLE:
• Standard periodization possible for most athletes
• Full threshold development achievable
• Can include proper taper

12+ WEEKS AVAILABLE:
• Full periodization with base, build, peak, and taper phases
• Can include tune-up race

────────────────────────────────────────────────────────────────────────────────
SECTION 13: TAPER AND RACE WEEK
────────────────────────────────────────────────────────────────────────────────

Half marathon taper is shorter and less aggressive than marathon taper.

TAPER STRUCTURE:

• Length: 10–14 days (shorter than marathon)
• Volume reduction: 30–40% in race week
• Final long run: 10–12 days before race (14–16 km)
• Final quality session: 5–7 days before race (short and sharp)

TAPER PRINCIPLES:

• Reduce volume but maintain some intensity
• Final threshold session 5–7 days before race (short: 2–3 × 1 km)
• Light strides 2–3 days before race
• Keep legs feeling sharp, not flat
• No gym in final 7–10 days
• The hay is in the barn — trust the training

RACE WEEK:

• 3–4 short easy runs
• 1–2 rest days
• Light strides 2 days before race
• Final shakeout day before race (20–30 min easy)

CALCULATING PACES:

• Half Marathon Pace = goal time ÷ 21.0975 km (LOCKED to goal)
• Threshold ≈ half marathon pace minus 10–15 sec/km
• Easy ≈ half marathon pace plus 60–90 sec/km
• Recovery ≈ easy pace plus 15–30 sec/km
• Steady = between easy and half marathon pace
• Interval ≈ threshold minus 15–20 sec/km
• Repetition ≈ interval minus 10–15 sec/km

If goal pace significantly exceeds current fitness:
• Note the gap in coaching overview
• Base TRAINING paces on current fitness
• Keep Half Marathon Pace in the table at goal pace (unchanged)
• Add Adjusted Training Pace for workout segments if needed

USING PACES:

• Every workout must specify target pace in min/km
• Do NOT use vague terms without the corresponding pace
• Example: "Easy run at 5:45/km" not just "Easy run"
